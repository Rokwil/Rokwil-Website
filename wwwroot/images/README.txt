INSTRUCTIONS FOR ADDING ACTUAL KEYSTONE PARK IMAGES
====================================================

1. Add your actual Keystone Park images to this folder (wwwroot/images/)

2. Required image files:
   - hero-keystone.jpg (for the homepage hero section)
     Recommended size: 1920x1080px or larger
   
   - keystone-park-main.jpg (for project showcase and projects page)
     Recommended size: 1200x800px or larger

3. Image format: JPG or PNG
   Quality: High resolution (at least 72 DPI for web)

4. Once you place the images in this folder with the exact names above,
   the website will automatically use them - no code changes needed!

5. The website is already configured to look for these images.
   Just add the files and refresh the page!
